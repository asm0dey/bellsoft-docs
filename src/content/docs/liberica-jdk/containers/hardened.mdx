`bellsoft/hardened-liberica-runtime-container` provides near-zero-CVE **nonroot**
and **distroless** Liberica JDK images (see [BellSoft Hardened Images](/bellsoft-docs/alpaquita/25-lts/hardened/overview/#hardened-image-types)).
Tag pattern:

```text
[jdk type]-[java version]-[crac]-[cds][-nonroot|-distroless]-[libc type]
```

| Example tag | Meaning |
| --- | --- |
| `jdk-17-nonroot-glibc` | nonroot JDK 17, glibc |
| `jre-21-crac-distroless-glibc` | distroless JRE 21 with CRaC, glibc |

<TagPicker mode="hardened" repos="hardened-liberica-runtime-container" />

Multi-stage build — compile in a nonroot image, run in distroless:

```dockerfile
FROM bellsoft/hardened-liberica-runtime-container:jdk-11-nonroot-glibc AS builder

COPY <<'EOF' HelloBellsoft.java
public class HelloBellsoft {
    public static void main(String[] args) {
        System.out.println("Hello BellSoft!");
    }
}
EOF

RUN javac HelloBellsoft.java

FROM bellsoft/hardened-liberica-runtime-container:jre-11-distroless-glibc

COPY --from=builder /home/appuser/HelloBellsoft.class .

ENTRYPOINT [ "/usr/bin/java" ]
CMD [ "HelloBellsoft"]
```
