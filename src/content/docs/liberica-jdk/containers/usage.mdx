## Pull

```bash
docker pull bellsoft/liberica-runtime-container:jdk-all-21-glibc
```

## Registries

Liberica runtime container images are published on all major container
registries. The repository name is the same across registries; only the
pull-URL prefix differs.

| Registry                     | Pull-URL prefix                                          | Authentication |
|------------------------------|----------------------------------------------------------|----------------|
| Docker Hub                   | `bellsoft/<repository>:<tag>`                            | No             |
| GitHub Container Registry    | `ghcr.io/bell-sw/<repository>:<tag>`                     | No             |
| Microsoft Container Registry | `bellsoft.azurecr.io/<repository>:<tag>`                 | No             |
| Google Container Registry    | `us-docker.pkg.dev/bellsoft/gcr.io/<repository>:<tag>`   | Yes            |
| Amazon ECR                   | `public.ecr.aws/bellsoft/<repository>:<tag>`             | No             |

<Aside type="note">
Pulling from Google Container Registry requires authentication. Run `gcloud auth configure-docker` before pulling.
</Aside>

The following examples pull the same image from each registry:


</TabItem>

</TabItem>

## Run

```bash
docker run --rm bellsoft/liberica-runtime-container:jdk-all-21-glibc java -version
```

## Build your app on top

```dockerfile
FROM bellsoft/liberica-runtime-container:jre-21-slim-glibc
COPY target/app.jar /app/app.jar
ENTRYPOINT ["java", "-jar", "/app/app.jar"]
```
